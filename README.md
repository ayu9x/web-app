# 🛠️ Tools Used in this Project

- **GitHub Actions**: For automated CI/CD pipeline and deployment.
- **Docker**: For containerization of services.
- **Docker Compose**: For orchestrating the multi-tier application.
- **Nginx**: As a high-performance reverse proxy.
- **Node.js (Express)**: For the backend API.
- **React (Vite)**: For the frontend user interface.
- **PostgreSQL**: For persistent data storage.
- **AWS EC2**: As the production hosting environment.
